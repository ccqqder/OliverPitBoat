## Python 點線面

> 你的隨身 Python 離線家教

Python 點線面以「點、線、面」三個維度構建 Python 學習體驗，從選擇題到程式碼排列，再到 AI 互動教學，系統性地掌握 Python。

![練習模式選擇](/images/screenshots/python-dimensions/practice.png)
*三種練習維度：「點」基礎選擇、「線」語法填空、「面」邏輯排序，循序漸進掌握 Python*

### 核心特色

- **三維訓練模式** — 選擇題（MCQ）、填空題（Cloze）、程式碼排列（Parsons）

![填空題](/images/screenshots/python-dimensions/cloze.png)
*「線」語法填空：從下拉選單中選擇正確的方法名稱，題目涵蓋 TQC+ 認證範圍*

![程式碼排序](/images/screenshots/python-dimensions/parsons.png)
*「面」邏輯排序：拖曳程式碼區塊排列正確順序，訓練完整的程式邏輯思維*

- **裝置端 AI 家教** — 搭配 RAG 技術，提供精準的 Python 知識問答

![AI 助教](/images/screenshots/python-dimensions/ai-chat.png)
*AI 助教：完全離線的 Python 問答，搭配 RAG 技術從題庫中檢索相關知識*

- **內建 Python 3.13 執行環境** — 直接在 App 內撰寫並執行 Python 程式碼

![Python 遊樂場](/images/screenshots/python-dimensions/playground.png)
*Python 遊樂場：內建程式碼編輯器與範本，可直接執行程式碼並查看輸出結果*
- **300+ 題庫** — 涵蓋 PCEP、TQC+、CPE 等認證考試
- **雷達圖分析** — 視覺化呈現你的各項能力分布
- **14 個可下載 AI 模型** — 依需求選擇最適合的模型
- **日式文具美學** — 以日本文具為靈感的介面設計
- **不收集任何使用者資料** — 所有運算在裝置端完成

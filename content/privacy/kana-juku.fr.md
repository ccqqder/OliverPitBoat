## 1. Vue d'ensemble

Kana Juku, développé par QQder339, est une application d'apprentissage des kana japonais conçue pour les locuteurs natifs de chinois.

**En bref : nous ne collectons, ne stockons ni ne transmettons AUCUNE de vos données personnelles vers des serveurs externes.**

## 2. Données que nous ne collectons PAS

Cette application ne collecte pas :

- Données personnellement identifiables (nom, e-mail, numéro de téléphone)
- Données de localisation
- Identifiants d'appareil
- Données d'analyse d'usage ou de suivi

## 3. Données stockées localement

Les données suivantes sont stockées strictement sur votre appareil et ne sont jamais transmises à l'extérieur :

- **Progression d'apprentissage** : suivi de votre statut d'apprentissage des kana
- **Réglages utilisateur** : sauvegarde de vos préférences
- **Saisie manuscrite** : traitée en mémoire en temps réel et supprimée immédiatement ; aucun fichier n'est enregistré
- **Données du widget** : utilise le mécanisme de conteneur partagé iOS pour afficher les kana dans les widgets de l'écran d'accueil (local uniquement)

## 4. Fonctionnalités IA hors ligne

Toutes les fonctionnalités IA fonctionnent entièrement hors ligne :

- **Reconnaissance de l'écriture manuscrite** : utilise des modèles d'apprentissage automatique sur l'appareil ; tout le traitement est local
- **Synthèse vocale** : utilise des ressources audio pré-téléchargées
- **Assistance IA** : utilise des modèles de langage large locaux (LLM) ; l'inférence est effectuée sur l'appareil sans téléchargement de données

## 5. Services tiers

Cette application n'utilise **AUCUN** framework d'analyse ou de publicité tiers (No Google Analytics, No Facebook SDK, No Ads).

## 6. Accès réseau

L'accès réseau est limité à :

- **Téléchargement des modèles IA** (optionnel) : se connecte uniquement lorsque vous choisissez explicitement de télécharger les ressources du modèle local
- **Liens externes** : ouvre le navigateur lorsque vous appuyez sur « Noter sur l'App Store » ou « Politique de confidentialité » ; ouvre le navigateur lorsque vous utilisez la fonction « Rechercher sur le Web » après traduction/reconnaissance

En dehors de ce qui précède, l'application n'initie pas de connexions réseau.
